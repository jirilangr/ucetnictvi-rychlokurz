# Nasazení na Vercel – jeden poslední krok

Repozitář je na GitHubu: **https://github.com/jirilangr/ucetnictvi-rychlokurz**

## Odkaz na import ve Vercelu

Klikni na tento odkaz (přihlas se k Vercelu přes GitHub, pokud ještě nejsi):

**[Import projektu do Vercelu →](https://vercel.com/new/import?repository-url=https%3A%2F%2Fgithub.com%2Fjirilangr%2Fucetnictvi-rychlokurz)**

1. Vercel načte repozitář `jirilangr/ucetnictvi-rychlokurz`.
2. **Root Directory** nech prázdné (nebo zadej `.`).
3. Klikni **Deploy**.
4. Po dokončení dostaneš adresu typu:  
   `https://ucetnictvi-rychlokurz-xxx.vercel.app`

## Odkazy po nasazení

- **Úvodní stránka:** `https://tvoje-url.vercel.app/`
- **Kurz:** `https://tvoje-url.vercel.app/ucetnictvi-rychlokurz.html`

(Alternativa: přihlas se na [vercel.com](https://vercel.com) → Add New → Project → vyber repozitář `ucetnictvi-rychlokurz` → Deploy.)
